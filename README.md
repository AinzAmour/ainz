# Portfolio Website

This is a personal portfolio website built with HTML, CSS, and JavaScript, showcasing cybersecurity skills and projects.

## Features

*   **Single-page layout:** All sections are on a single page for easy navigation.
*   **Responsive design:** The website is optimized for viewing on different screen sizes, from mobile devices to desktops.
*   **Interactive elements:** The website includes a matrix rain effect, a typewriter effect, and a glitch animation.
*   **Accessibility:** Includes ARIA labels, semantic HTML, and keyboard navigation support.
*   **Dark theme:** A sleek, hacker-inspired dark theme with neon accents.

## Getting Started

To run the project locally, simply open the `index.html` file in a web browser. No server is required.

## Deployment to GitHub Pages

1. **Create a GitHub Repository:**
   - Go to GitHub and create a new repository (e.g., `yourusername.github.io` for a user site, or any name for a project site).
   - If using `yourusername.github.io`, GitHub will automatically serve it as your personal site.

2. **Upload Files:**
   - Clone the repository to your local machine.
   - Copy all files from this project into the repository folder.
   - Commit and push the changes to GitHub.

3. **Enable GitHub Pages:**
   - Go to your repository on GitHub.
   - Navigate to **Settings > Pages**.
   - Under **Source**, select **Deploy from a branch**.
   - Choose the `main` (or `master`) branch and the root folder (`/`).
   - Click **Save**.

4. **Access Your Site:**
   - After a few minutes, your site will be live at `https://yourusername.github.io` (for user sites) or `https://yourusername.github.io/repository-name` (for project sites).

## File Structure

*   `index.html`: The main HTML file containing the structure and content.
*   `style.css`: The CSS file for styling the website, including themes and responsive design.
*   `script.js`: The JavaScript file for interactivity, animations, and theme switching.
*   `resume.pdf`: The downloadable resume file.

## Customization

To customize the website, you can edit the following files:

*   **`index.html`**: To change the content of the website, such as the text, images, and links.
*   **`style.css`**: To change the appearance of the website, such as the colors, fonts, and layout.
*   **`script.js`**: To change the behavior of the website, such as the animations and interactive elements.
*   **`resume.pdf`**: To update the resume.

## Technologies Used

- JavaScript (ES6+)
- FontAwesome for icons
